TraceCraft Demo App

Use this local app to generate controlled failures and performance issues while the TraceCraft browser extension records clicks, network activity, console output, and server-side logs.

500 Error

Triggers a server exception with a stack trace and a 500 API response.

Slow Endpoint

Simulates a sluggish downstream call so the browser captures a slow network request.

Malformed JSON

Returns invalid JSON so the frontend throws a parsing error after a nominal 200 response.

Validation Error

Posts bad input to trigger a logged 400 validation error on the server.

Latest Result

Press a button to generate activity for TraceCraft.
